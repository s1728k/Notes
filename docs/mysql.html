MySql Commands

Command explanation
mysqldump -u root -p database_name > database_name.sql exports the database into query sheet
mysql -u root -p database_name < database_name.sql imports the database queries into existing database
show databases; lists all databases.
create database database_name; creates new database.
use database_name; selects database.
select user, host from mysql.user; lists the users @ hostname in mysql account
create user 'user_name'@'localhost' identified BY 'password'; creates the user 'user_name' in mysql account
alter user 'user_name'@'localhost' identified BY 'password'; changes the password for the user 'user_name' in mysql account
drop user 'user_name'@'localhost'; deletes the user 'user_name' in mysql account
grant all privileges on database_name.* to 'user_name'@'localhost'; gives all permissions to user on database.
flush privileges; execute this after granting permission to user on database.
show tables; lists all tables in selected database.
php artisan config:clear
